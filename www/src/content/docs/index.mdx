## Why SitePod?

<CardGrid>
  <Card title="Single Binary" icon="rocket">
    Caddy + PocketBase + SQLite. Download, run, done. No Docker, no dependencies.
  </Card>
  <Card title="Instant Rollback" icon="seti:clock">
    Every deploy is an immutable snapshot. Roll back to any version in under a second. No rebuild.
  </Card>
  <Card title="Zero Config" icon="seti:config">
    Detects your build directory, generates subdomains, handles SSL. Just run `deploy`.
  </Card>
  <Card title="Self-Hosted" icon="seti:lock">
    Your server, your data. Runs on any $5/mo VPS.
  </Card>
</CardGrid>

## How It Works

<div style="background: linear-gradient(135deg, rgba(16, 185, 129, 0.05) 0%, rgba(6, 182, 212, 0.05) 100%); border: 1px solid rgba(16, 185, 129, 0.2); border-radius: 12px; padding: 2rem; margin: 2rem 0;">

```
                    DEPLOY                          SERVE
                      │                               │
     ┌────────────────┼────────────────┐              │
     │                │                │              │
     ▼                ▼                ▼              │
┌─────────┐     ┌─────────┐     ┌─────────┐          │
│ Pod v1  │     │ Pod v2  │     │ Pod v3  │    ◄─────┤
│ ─────── │     │ ─────── │     │ ─────── │          │
│ snap-   │     │ snap-   │     │ snap-   │          │
│ shot    │     │ shot    │     │ shot    │          │
└─────────┘     └─────────┘     └─────────┘          │
                      ▲               ▲               │
                      │               │               │
                ┌─────┴───┐     ┌─────┴───┐          │
                │  beta   │     │  prod   │    ◄─────┘
                │  (ref)  │     │  (ref)  │
                └─────────┘     └─────────┘

     Rollback = move pointer (instant, no rebuild)
```

</div>
